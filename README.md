# 🎓 Aprenda HTML - Site de Estudos

Um site educativo completo para aprender HTML do básico ao avançado.

## 📁 Estrutura do Projeto

```
site-html/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # JavaScript para interatividade
├── README.md           # Este arquivo
└── videos/             # Roteiros para vídeos educativos
    ├── roteiro-video-1-introducao.md
    ├── roteiro-video-2-tags-essenciais.md
    ├── roteiro-video-3-formularios.md
    ├── roteiro-video-4-tabelas-e-layout.md
    └── guia-gravacao.md
```

## 🚀 Como Usar

1. **Abrir o site**: Basta abrir o arquivo `index.html` em qualquer navegador
2. **Estudar**: Navegue pelas seções de conteúdo
3. **Praticar**: Use o editor em tempo real na seção de prática
4. **Vídeos**: Consulte os roteiros na pasta `videos/` para gravar aulas

## 📚 Conteúdo do Site

### Seções:
- **Início**: Introdução ao HTML
- **Vídeos**: Roteiros para aulas em vídeo
- **Básico**: Estrutura e tags fundamentais
- **Elementos**: Texto, listas, links, imagens, tabelas
- **Formulários**: Inputs e validação
- **Prática**: Editor em tempo real + desafios

## 🎥 Roteiros de Vídeo

| Aula | Título | Duração | Tópicos |
|------|--------|---------|---------|
| 1 | Introdução ao HTML | 5-7 min | O que é HTML, web, primeiro arquivo |
| 2 | Tags HTML Essenciais | 10-12 min | H1-H6, p, a, img, div, span |
| 3 | Formulários HTML | 10-12 min | Form, inputs, validação |
| 4 | Tabelas e Layout | 8-10 min | Tables, HTML semântico |
